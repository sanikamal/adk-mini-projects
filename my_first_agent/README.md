# My First Agent

This is a starter agent project generated with the Google Agent Development Kit (ADK).

## Overview
This project sets up a basic `root_agent` using the `gemini-2.5-flash` model. It serves as a starting point to learn how to:
- Define an agent's instruction and identity.
- Manage environment variables using a `.env` file.
- Run the agent locally using the ADK web interface.

## Quick Start
1. Ensure your virtual environment is activated.
2. Ensure you have your `.env` configured with `GOOGLE_API_KEY` or Vertex AI credentials.
3. Start the agent:
   ```shell
   adk web
   ```
4. Navigate to the provided local URL (typically `http://localhost:8000`) to interact with your agent.
